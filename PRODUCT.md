# Product

## Register

product

## Users

Mac users who repeatedly jump between apps, web platforms, and keyboard-driven actions while working. They use the app in the middle of other workflows, so the interface should help them configure shortcuts quickly and trust that the floating launcher will behave predictably.

## Product Purpose

Float Launcher is a local shortcut command surface. It manages a floating button that can open apps or web pages, run configured automations such as Feishu Minutes recording, and trigger saved hotkey combinations.

## Brand Personality

Quiet, tactile, capable.

## Anti-references

Avoid a generic SaaS dashboard, marketing-style hero screens, decorative gradients, heavy glassmorphism, and playful controls that obscure standard settings behavior.

## Design Principles

- Put the current floating button state first.
- Make launch shortcuts and hotkey shortcuts easy to scan and distinguish.
- Keep configuration inline, local, and reversible.
- Use motion and color only to clarify state or direct attention.
- Preserve macOS familiarity in controls, spacing, and labels.

## Accessibility & Inclusion

Target WCAG AA contrast for text and controls. Every icon-only control needs an accessible label. Motion should be brief and must respect reduced-motion preferences.
